A PHP library to parse a Standard Algebraic Notation (SAN).

## Installation

Via composer

```
composer require chesszebra/standard-algebraic-notation
```

## Usage

```php
$notation = new ChessZebra\StandardAlgebraicNotation\Notation('e8=Q');
```
